# Termo_furnace
Термо печка

Режимы кнопки:
- двойной клик = включается экран OLED на 22 сек
- удержание = увеличивается значение температуры max
- клик = значение температуры min=max
- клик+удержание = увеличивается значение часов нагрева nHour
